## 🛠 Methodological Pillar: The "From Scratch" Philosophy
Every architecture in this repository is deconstructed into its fundamental components:
1. **Mathematical Derivation:** Understanding the calculus, linear algebra, and probability distributions involved.
2. **NumPy Implementation:** Building the logic without high-level libraries to ensure deep algorithmic mastery.
3. **Production Deployment:** Scaling concepts using PyTorch, Scikit-Learn, and modern UI frameworks (Plotly Dash).
